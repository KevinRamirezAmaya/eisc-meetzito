Dado que este proyecto utiliza un **monorepositorio**, es necesario acceder a las carpetas correspondientes para ejecutar tanto el **backend** como el **frontend**.

---

### Backend
Para ejecutar el backend, entra en su carpeta y ejecuta el servidor:

```bash
cd eisc-chat
npm install
npm run dev
```

### Frontend
Para ejecutar el frontend, entra en su carpeta y ejecuta el servidor:

```bash
cd eisc-meet
npm install
npm run dev
```
